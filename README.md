# Zenn CLI

* [📘 How to use](https://zenn.dev/zenn/articles/zenn-cli-guide)


## 自分用メモ

```shell
# 新しい記事の生成
npx zenn new:article --slug 記事ファイル名(小文字英数字、ハイフン、アンダースコアのみ)

# ローカルホストが立ち上がりブラウザでPreviewできる。
npx zenn preview
```
